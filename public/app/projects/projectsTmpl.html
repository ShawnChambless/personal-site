Recent Projects

Grain Billd V.1

Grain Billd was my choice for the "personal project" for DevMountain. The personal project was meant to be a full stack web app utilizing all of the MEAN Stack technologies that we learned at DevMountain. Grain Billd is build with AngularJS, Node.js, Express, MongoDB, and Mongoose. User authentication is accomplished via the Passport Local Auth Strategy. I really enjoyed building Grain Billd due to the overwhelming need for a better way to calculate beer recipes, and the lack of a single place to research, calculate, and save the recipes.

Grain Billd V.2

After finishing the cohort, I went back and looked at the Grain Billd code, and was not satisfied with the quality. I decided to redo the code more functionally, and make the UI more user friendly (very much a work in progress). I had no idea how fun functional code could be, and how much it improves readability and debugging.

DevMountain Resources

The DevMountain Resources app was also a full stack MEAN project, but with a group this time. It was built with all of the same technologies as Grain Billd was, and all of the learning curve of using git as a group. This is where I really solidified my passion for web development. Developing as a group is the pinnacle of fun at work for me. The project was contracted to my group by DevMountain, and purchased by the school. We are still adding features, and refactoring the code.
